# Projeto-Escola-Virtual
Pequena aplicação que permite o cadastro de pessoas em um banco de dados.

O código foi desenvolvido pelo ministrador do curso "Desenvolvendo um Projeto Completo Python com Estruturas de Dados" da Escola Virtual da Fundação Bradesco. Fiz apenas algumas adaptações necessárias, como por exemplo um startDB() em uma linha do código original que impedia o funcionamento correto da aplicação.
Durante o minicurso, foi explicado o funcionamento e as origens do SQLite, seus usos mais comuns, como realizar consultas, criar tabelas, fazer filtragens, e algumas outras funcionalidades. Ao final do curso, esse código foi passado para o aluno para que ele pudesse visualzar na prática a integração do SQLite com Python. Para executar, basta executar o arquivo aplicação.py.
